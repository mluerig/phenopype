How to run tutorials and examples using Jupyter Notebook¶

The tutorials are written as jupyter notebooks - browser based Python kernels to run, document, and visualize code (https://jupyter.org/). Below is a step by step guide to get started with the tutorials and examples. If you installed Python using Anaconda, it is possible that jupyter is already installed (jupyter --version in the terminal should return a version number) - in this case you can skip step 3.

  1. Install phenopype: follow the installation instructions.

  2. Download and unpack the phenopype repository from github

  3. Install jupyter notebook to your environment*: pip install jupyter notebook

  4. Open a terminal and go to the tutorial folder in the unpacked phenopype repository (e.g. ../downloads/phenopype-master/tutorials)

  5. Start the notebooks with jupyter notebook and click on one of the tutorial files.

  6. Run the code cell by cell

Warning

* Make sure you install jupyter to your specific environment (i.e. activate first using conda activate "my-env". If jupyter is not installed, running jupyter notebook will fall back on the conda base environment where phenopype may not be installed (this is a common error source).
